## Accounting method

This is our method to easily split bills!

ex :

```
Bills of user1 : b1, b2, b3
Bills of user2 : b4, b5, b6

Sum user1 : s1 = b1 + b2 + b3
Sum user2 : s2 = b4 + b5 + b6

If s1 > s2, user2 gives ((s1 - s2) / 2)$ to user1
```
